# Predicción de Churn en Telecomunicaciones - Part-1
Descripción
Este proyecto analiza los datos de clientes de TelecomX LATAM para identificar patrones asociados a la evasión de clientes (Churn). Utiliza técnicas de limpieza, transformación y análisis exploratorio de datos, complementadas con visualizaciones, para obtener insights que ayuden a reducir la tasa de cancelación.

Características principales
Importación y procesamiento de datos desde formato JSON.
Limpieza y tratamiento de datos (tipos, nulos, estandarización).
Análisis exploratorio con visualizaciones (gráficos de barras, histogramas, comparativas).
Identificación de variables más influyentes en la evasión.
Recomendaciones estratégicas basadas en el análisis.

Tecnologías utilizadas
-Python 3.x
-Pandas – Manipulación de datos.
-Matplotlib / Seaborn – Visualización de datos.
-Jupyter Notebook – Desarrollo y documentación interactiva.

Uso
-Ejecuta las celdas del notebook para reproducir el análisis.
-Modifica las rutas de los datos si utilizas un archivo distinto.
-Genera tus propias visualizaciones para explorar otras variables.

Resultados principales
Contratos mensuales → Mayor riesgo de Churn.
Clientes con facturación alta → Mayor probabilidad de cancelación.
